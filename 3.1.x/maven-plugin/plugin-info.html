Plugin Documentation

Goals available for this plugin:

Goal Description
liquigraph:dry-run Simulates Liquigraph execution and persists results in ${project.build.directory}/output.cypher.
liquigraph:help Display help information on liquigraph-maven-plugin.
Call mvn liquigraph:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
liquigraph:run Triggers Liquigraph execution.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.liquigraph</groupId>
          <artifactId>liquigraph-maven-plugin</artifactId>
          <version>3.1.1</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.liquigraph</groupId>
        <artifactId>liquigraph-maven-plugin</artifactId>
        <version>3.1.1</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
